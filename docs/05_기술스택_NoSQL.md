# 05. 기술 스택 - NoSQL 데이터베이스 (MongoDB)

> 상세 내용은 `프로젝트_필수요소.md` 참조

## 선정 결과: MongoDB

### 주요 선정 이유

1. **가변 스키마 완벽 지원** ✨
   - 블루투스 있는 키보드 / 없는 키보드
   - 각 제품마다 다른 필드 구조 OK

2. **트랜잭션 지원**
   - 답변 승인 시 여러 컬렉션 동시 업데이트
   
3. **강력한 쿼리 & 집계**
   - 복잡한 조건 검색
   - 비즈니스 인사이트 추출
